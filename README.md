# 🧮 Smart Calculator using Python

A simple and advanced console-based calculator application built using Python.  
This project performs basic and advanced mathematical operations with calculation history support.

## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- √ Square Root Calculation
- 🔢 Power Calculation
- 📊 Percentage Calculation
- 🕒 Calculation History
- 🔁 Continuous Calculation
- ⚠️ Error Handling

## 🛠️ Technology Used

- Python
- Math Module

## 📂 Project Structure

```
Smart-Calculator-Python/

│── calculator.py
│── README.md
│── requirements.txt
```

## ▶️ How to Run

1. Clone the repository

```
git clone <your-repository-link>
```

2. Open the project folder

```
cd Smart-Calculator-Python
```

3. Run the program

```
python calculator.py
```
